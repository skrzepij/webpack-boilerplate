# README #


## How to run ##

1. Clone repo
2. Run these commends in terminal:
    1. `npm install`
    2. `npm run dev`


There are other npm scripts available in `package.json`  i.e.:
```
#!bash

npm run serv
npm run clean
npm run prod
```

As mentioned before, all these scripts are available in the `package.json` file
